Surgimento da Internet

A Internet teve origem no contexto da Guerra Fria, como um projeto militar dos Estados Unidos chamado ARPANET (Advanced Research Projects Agency Network),
desenvolvido no final da década de 1960 pela ARPA (agora conhecida como DARPA - Defense Advanced Research Projects Agency). A ideia era criar uma rede de
comunicação robusta e descentralizada que permitisse a troca de informações entre instituições de pesquisa e militares, garantindo a resiliência em caso de ataques inimigos.

A primeira conexão bem-sucedida na ARPANET ocorreu em 29 de outubro de 1969, quando uma mensagem foi enviada de um computador na
Universidade da Califórnia em Los Angeles (UCLA) para um computador na Universidade de Stanford, percorrendo uma rede de quatro nós.

Nos anos seguintes, a ARPANET foi crescendo e conectando mais instituições acadêmicas e militares nos Estados Unidos. Em 1983, a ARPANET mudou
o protocolo de comunicação de NCP (Network Control Protocol) para o TCP/IP (Transmission Control Protocol/Internet Protocol), que se tornou a base da arquitetura da Internet moderna.

Surgimento do VsCode

Topo da página

O Visual Studio Code (VSCode) foi criado pela Microsoft e lançado pela primeira vez em abril de 2015. Ele foi desenvolvido com o objetivo de ser um
editor de código-fonte leve, rápido, altamente extensível e multiplataforma, adequado para desenvolvimento web e de aplicativos.

A ideia de criar o VSCode surgiu como uma resposta ao crescimento da popularidade de editores de código mais leves, como o Sublime Text e o Atom.
Esses editores ganharam destaque por sua velocidade e flexibilidade, especialmente no contexto do desenvolvimento web, onde a agilidade e o suporte a várias linguagens de programação são cruciais.
Em vez de criar um IDE (Integrated Development Environment) completo como o Visual Studio, que é mais pesado e voltado para desenvolvimento em várias
linguagens e plataformas, a Microsoft optou por desenvolver um editor de código mais enxuto e focado em extensibilidade.

O Visual Studio Code foi construído com base na tecnologia Electron, que permite o desenvolvimento de aplicativos de desktop multiplataforma usando tecnologias web como HTML, CSS e JavaScript.

Funcionamento do HTML

Topo da página

O HTML (Hypertext Markup Language) é uma linguagem de marcação usada para criar páginas web. Ele funciona através da estruturação do conteúdo da página utilizando elementos (tags)
que definem diferentes partes, como texto, imagens e links. As tags podem conter atributos que modificam o comportamento dos elementos.

Quando um navegador processa um documento HTML, ele interpreta as tags e seus atributos para renderizar a página, exibindo o conteúdo ao usuário de forma formatada e funcional.
Além disso, o HTML oferece elementos semânticos que fornecem significado ao conteúdo, facilitando a compreensão da estrutura da página por motores de busca e leitores de tela.
O HTML é frequentemente combinado com outras tecnologias, como CSS e JavaScript, para adicionar estilos e interatividade às páginas web.

Funcionamento de tags

Topo da página

No HTML, as tags são elementos fundamentais que definem a estrutura e o conteúdo de uma página web. Elas são escritas entre colchetes angulares ("<" e ">")
e podem conter atributos que modificam o comportamento ou aparência do elemento.

Cada tag tem uma função específica, como definir títulos, parágrafos, imagens, links, formulários e muito mais. As tags podem ser aninhadas para criar hierarquias de conteúdo,
e o navegador da web interpreta essas tags para renderizar a página, exibindo o conteúdo ao usuário de forma formatada e funcional.
O uso adequado de elementos semânticos ajuda na compreensão da estrutura da página, tanto para humanos quanto para máquinas, facilitando a acessibilidade e otimização para motores de busca.
